Passo 2

Passo 3

Esta opção é suficiente para a entrega do vídeo. As opções B e C são para mostrar integração real (nota máxima).

Opção B Node-RED com dados simulados (sem Pico W)

Instale o Node-RED e importe o flow. Os botões "inject" simulam o Pico W sem precisar de hardware.

1 — Instalar Node-RED (uma vez)
npm install -g --unsafe-perm node-red
node-red

Acesse: http://localhost:1880

2 — Instalar plugins necessários

No Node-RED, menu superior direito → Manage palette → aba Install:

  • node-red-dashboard — interface visual
  • node-red-contrib-influxdb — integração InfluxDB (opcional)
3 — Importar o flow

Menu → Import → selecione o arquivo node_red_flow.json → Import → Deploy

4 — Testar sem hardware

No flow importado, clique nos botões inject:
• "Teste: SEGURO" → sistema normal
• "Teste: AVISO pessoa" → pessoa detectada
• "Teste: ALERTA" → alerta completo (temp + pessoa)

Acesse o dashboard em: http://localhost:1880/ui

Opção C Pipeline completo com Pico W físico

1 — Instalar Mosquitto (MQTT Broker)

Mac: brew install mosquitto && brew services start mosquitto
Windows: Download em mosquitto.org → instalar como serviço

2 — Configurar o firmware do Pico W

Abra main_hardware.py e ajuste:

WIFI_SSID   = "nome_da_sua_rede"
WIFI_PASS   = "sua_senha_wifi"
MQTT_BROKER = "192.168.x.x"   # IP do seu Mac/PC (ifconfig ou ipconfig)

Copie main_hardware.py e ssd1306.py para o Pico W via Thonny ou mpremote:

mpremote cp main_hardware.py :main.py
mpremote cp ssd1306.py :ssd1306.py
3 — Conectar Node-RED ao Mosquitto

No flow importado, clique duas vezes no nó "MQTT — Forno A":
• Server: localhost · Port: 1883
• Clique em Update → Deploy

4 — Verificar pipeline

Ligue o Pico W → ele conecta ao Wi-Fi → publica no Mosquitto → Node-RED recebe → dashboard atualiza.

Verifique no terminal do Mosquitto:

mosquitto_sub -h localhost -t "quantumfinance/#" -v
5 — (Opcional) InfluxDB + Grafana

Mac:

brew install influxdb grafana
brew services start influxdb
brew services start grafana

InfluxDB: http://localhost:8086 — crie o bucket quantumfinance
Grafana: http://localhost:3000 — adicione InfluxDB como datasource → crie dashboard
